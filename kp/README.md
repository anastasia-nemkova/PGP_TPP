# Обратная трассировка лучей (Ray Tracing) на GPU


![Тетраэдр, октаэдр, додекаэдр](./animations/408_Немкова_Анастасия_Романовна_3.gif)

Команды для запуска

#### Компиляция
```
make clean
make
```

### Показать конфигурацию по умолчанию
```
./kp --default
```

### Запуск на CPU
```
./kp --cpu - чтение параметров из stdout
./kp --cpu < config.txt - чтение параметров из файла
```

# Запуск на GPU (по умолчанию)
```
./kp || ./kp --gpu- чтение параметров из stdout
./kp < config.txt || ./kp --gpu < config.txt- чтение параметров из файла
```

# Cкрипт для конвертации рендеров трассировки лучей
```
chmod +x run_converter.sh
./run_converter.sh
```
